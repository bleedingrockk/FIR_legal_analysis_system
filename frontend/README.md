# FIR Legal Analysis - Frontend

Modern React + TypeScript + Tailwind CSS frontend for the FIR Legal Analysis application.

## Development

```bash
npm install
npm run dev
```

The dev server runs on http://localhost:3000 and proxies API requests to http://localhost:8000

## Building for Production

```bash
npm run build
```

This builds the React app and outputs all static files to `../backend/static`, which the FastAPI backend serves.
